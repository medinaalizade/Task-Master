# 🎯 Task Master

A beautiful, full-stack todo application with a modern glassmorphism UI, dark mode, progress tracking, and cloud persistence. Built with React, Node.js, Express, and MongoDB Atlas.

## ✨ Features

- ✅ **Create, complete & delete** tasks
- 📊 **Progress bar** with completion percentage
- 🌙 **Dark / Light mode** toggle (saves preference)
- 🗑️ **Clear All** tasks with confirmation
- 💎 **Glassmorphism design** with smooth animations
- 📱 **Responsive** — works on mobile & desktop
- ☁️ **Cloud persistence** — data survives refresh, syncs across devices

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | React 19 + Vite |
| **Styling** | Custom CSS (Glassmorphism, CSS Variables) |
| **State** | React Hooks (useState, useEffect) |
| **HTTP** | Axios |
| **Backend** | Node.js + Express 5 |
| **Database** | MongoDB Atlas |
| **Deployment** | Render |
